Frontend არის ვებ-დეველოპმენტის სფერო, რომელიც პასუხისმგებელია მომხმარებლის ინტერფეისის შექმნაზე.

What is Frontend?

Frontend არის ყველა ის ნაწილი, რაც მომხმარებელს სჩანს და interacts როგორც ვებ-გვერდი. Frontend-ის დეველოპმენტში შედის HTML, CSS და JavaScript. HTML გამოიყენება სტრუქტურისთვის, CSS – დიზაინისთვის, ხოლო JavaScript – დინამიური ფუნქციონალობისთვის.

HTML

HTML (HyperText Markup Language) არის საწყისი ენა, რომლითაც ვებ-გვერდების სტრუქტურას ქმნიან. HTML საშუალებას იძლევა, შექმნათ ტექსტი, სურათები, ვიდეოები, ფორმები და სხვა მრავალი ელემენტი, რომლებიც ვებ-გვერდის შემადგენლობაა.

History

HTML შეიქმნა 1990 წელს, როდესაც ტიმ ბერნერს-ლი წამოაყენა ვებ-გვერდების განვითარება. მას შემდეგ HTML მუდმივად ვითარდებოდა და დაემატა ახალი ეტიკეტები და ატრიბუტები.

CSS

CSS (Cascading Style Sheets) არის ენა, რომელიც გამოიყენება ვებ-გვერდების დიზაინის შესაქმნელად. CSS საშუალებას იძლევა, რომ სტრუქტურულად შექმნილი HTML დოკუმენტი გამოიყურებოდეს უფრო ესთეტიკურად და გახდეს ადვილად გასაგები მომხმარებლებისთვის.

JavaScript

JavaScript არის სკრიპტინგის ენა, რომელიც გამოყენებულია ვებ-გვერდებზე ინტერქტიული ფუნქციების დამატებისთვის. ის გამოიყენება სხვადასხვა ოპერაციების, მაგალითად ფორმების ვალიდაციის, ანიმაციების, და დინამიური მონაცემების განახლებისთვის.
